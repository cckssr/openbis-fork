Welcome to openBIS Laboratory Notebook

Here is a quick guide how to navigate through this page.


You can find following entities in this notebook:

  • Space: a top-level organizational directory that may contain Projects, Collections and Objects. Only Instance Admin may create new spaces.
    • Home Space: space hinted as being owned or managed by the user who see it as home. (*)
  • Project: a directory under space level.
  • folder Folder: an entity with only "name" metadata property.
  • Entry: a basic entity, a blank page, with no pre-defined fields. Depending on the content, a different iconography may be applied:
    • subject Metadata Entry: entry with no files (metadata only).
    • attach_file Data Entry: entry with only data.
    • Metadata/Data Entry: entry with both data and metadata.
  • Dataset: immutable container for data files.

You can read more here

(*): Designation of a home space is configuration-specific, ask your admin if needed.
